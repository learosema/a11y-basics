# Assistive Tools

- Magnifiers, OS-built-in or special software, ZoomText
- [Screen readers](https://dequeuniversity.com/screenreaders/)
  - VoiceOver for Mac/iOS
  - Narrator for Windows
  - NVDA, JAWS
- Braille display
- alternative input devices for motor disabilities
- Closed Captions
- Transcripts
